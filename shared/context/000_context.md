# Shared — Context

Contexto geral comum da Previmed.

## Objetivo
Reunir o contexto de base sobre a Previmed que qualquer departamento, workspace ou agente precisa de conhecer.

## O que vai guardar aqui
- o que é a Previmed e o que faz;
- enquadramento geral da atividade;
- contexto partilhado entre departamentos e workspaces.

## Como os agentes vão usar
É leitura de **contexto mínimo**: o agente lê isto cedo para perceber o pedido antes de descer a um departamento/workspace específico.
