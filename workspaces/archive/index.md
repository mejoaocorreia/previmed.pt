# Workspaces — Archive

Arquivo de workspaces encerrados ou históricos.

## Objetivo
Guardar workspaces que já não estão ativos, sem os apagar, para manter histórico e contexto.

## O que vai guardar aqui
- workspaces concluídos ou descontinuados;
- notas sobre por que foram arquivados.

## Departamentos relacionados
Variável — conforme o workspace arquivado.

## Tools relacionadas
Variável — conforme o workspace arquivado.

## Como os agentes vão usar
O archive é a **última camada** de leitura: só consultar quando um workspace ativo remete para histórico.
