# Command: seo-audit

## Objetivo

Executar auditoria SEO controlada.

## Uso

Quando o utilizador pedir:
- auditoria SEO;
- revisão SEO;
- análise técnica;
- análise de indexação;
- plano SEO.

## Processo

1. Confirmar escopo.
2. Classificar risco.
3. Verificar se há produção/dados.
4. Chamar SEO Lead.
5. Chamar Technical SEO se houver técnica.
6. Usar Growth Quality Gate.
7. Persistir relatório se análise for grande.

## Output

- diagnóstico;
- evidência;
- prioridades;
- riscos;
- plano;
- ficheiros/páginas afetadas;
- próximos passos.

## Regra

Não alterar site.
Não alterar WordPress.
Não mudar indexação/URLs sem autorização.
