### Crawl / Indexacao

### WordPress

Plugin SEO, schema duplicado, archives, attachment pages, templates, cache, tema.

### Renderizacao

Conteudo no HTML/render, JS, links `<a href>`, lazy load, mobile.

### On-page

Titles, metas, headings, conteudo, imagens, alt text, links internos.

### Conteudo / Intencao

Intencao vs pagina, E-E-A-T, claims, prova, lacunas.

### Concorrencia / SERP

Padroes, page types, SERP features, gaps, oportunidades.

### Schema / Entidades

Schema presente, em falta, duplicado, enganador ou desalinhado.

### Local SEO

GBP, NAP, paginas locais, areas de servico, risco de doorway pages.

### Performance / CWV

LCP, INP, CLS, mobile, field vs lab data, templates afetados.

### AI Search / GEO

Entidades, conteudo citavel, respostas, visibilidade observada, sem promessas.
