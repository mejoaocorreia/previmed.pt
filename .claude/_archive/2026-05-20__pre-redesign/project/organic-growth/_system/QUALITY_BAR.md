# SEO Quality Bar

A fasquia SEO deste projeto é alta.

## Uma página SEO excelente deve ter

- intenção clara;
- público claro;
- proposta de valor;
- H1 único e descritivo;
- estrutura H2/H3 lógica;
- title forte;
- meta description persuasiva;
- conteúdo útil, completo e original;
- sinais de confiança;
- CTA claro;
- links internos relevantes;
- breadcrumbs;
- schema apropriado;
- performance aceitável;
- mobile excelente;
- acessibilidade básica;
- URL limpa;
- canonical correto;
- indexabilidade;
- ausência de conteúdo duplicado inútil;
- design que ajuda a leitura.

## Rejeitar se

- parece genérica;
- só troca palavras dos concorrentes;
- usa keyword stuffing;
- não responde à intenção;
- não tem CTA;
- não tem confiança;
- é lenta;
- está escondida atrás de JS desnecessário;
- tem headings usados apenas como estilo;
- tem schema falso;
- cria canibalização;
- é uma doorway page.

## Conteúdo people-first

O conteúdo deve ser feito para ajudar pessoas reais.
SEO deve ajudar motores de busca a entender conteúdo bom, não fabricar relevância artificial.
