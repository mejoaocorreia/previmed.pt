# SEO KPI Model

## KPIs principais

- organic clicks;
- impressions;
- CTR;
- average position;
- organic sessions;
- organic conversions/leads;
- indexed pages;
- pages with impressions;
- Core Web Vitals;
- non-brand visibility;
- local visibility;
- page-level conversion.

## KPIs por fase

### Fase 1 — Fundamentos
- indexabilidade;
- sitemap válido;
- titles/metas;
- Core Web Vitals críticos;
- páginas prioritárias publicadas.

### Fase 2 — Crescimento
- impressões por cluster;
- clicks por página;
- CTR improvements;
- novas queries;
- rankings top 20/top 10;
- links internos.

### Fase 3 — Conversão
- leads orgânicos;
- CTR para CTAs;
- formulários;
- chamadas/contactos;
- páginas de serviço com maior retorno.

## Regras

- Não avaliar SEO só por ranking.
- Separar brand e non-brand.
- Comparar períodos equivalentes.
- Não tirar conclusões sem amostra suficiente.
- Relacionar métricas com ações feitas.
