# SEO Content System

## Tipos de páginas

- Homepage
- Páginas de serviço
- Páginas por setor
- Páginas locais, quando justificadas
- Guias educativos
- FAQs
- Casos/credenciais
- Contactos/conversão
- Blog/artigos, se houver estratégia clara

## Brief obrigatório para página importante

- objetivo da página;
- público;
- intenção;
- keyword/tópico principal;
- tópicos secundários;
- proposta de valor;
- estrutura H1/H2/H3;
- perguntas respondidas;
- prova/confiança;
- CTA;
- links internos;
- schema recomendado;
- assets necessários;
- critérios de qualidade.

## Regras

- Não criar página sem intenção clara.
- Não criar conteúdo duplicado entre serviços.
- Não criar páginas locais vazias.
- Não publicar texto sem revisão humana.
- Não usar IA para gerar massa sem valor.
