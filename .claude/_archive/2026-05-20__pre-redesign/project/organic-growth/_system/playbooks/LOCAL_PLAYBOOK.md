# Local SEO Playbook

Usar quando o projeto tiver presença local, áreas de serviço ou intenção local.

## Verificar

- Google Business Profile;
- NAP consistente;
- morada/contactos no site;
- páginas locais justificadas;
- service areas;
- reviews;
- schema LocalBusiness;
- mapas;
- links para perfis oficiais;
- diretórios/citations.

## Regras

- Não criar doorway pages.
- Não inventar moradas.
- Não criar páginas locais sem conteúdo útil.
- Não alterar Google Business Profile sem autorização.
- Website e GBP devem contar a mesma história.
