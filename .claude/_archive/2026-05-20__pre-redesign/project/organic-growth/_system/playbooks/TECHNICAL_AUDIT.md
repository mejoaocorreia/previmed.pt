# Technical SEO Audit Framework

Verificar:

## Crawl / Index
- robots.txt;
- sitemap.xml;
- status codes;
- index/noindex;
- canonical;
- redirects;
- broken links;
- orphan pages;
- duplicate titles/descriptions;
- thin content;
- pagination;
- faceted URLs, se existirem.

## WordPress
- plugin SEO;
- schema duplicado;
- archives indexáveis desnecessárias;
- author/date archives;
- category/tag pages;
- media attachment pages;
- theme templates;
- enqueues;
- caching.

## Renderização
- conteúdo principal no HTML/renderizado;
- JS não bloqueia indexação;
- links reais em `<a href>`;
- lazy load não esconde conteúdo crítico.

## Performance
- LCP;
- INP;
- CLS;
- imagens;
- fontes;
- JS/CSS;
- cache;
- mobile.

## Structured data
- JSON-LD;
- Organization/LocalBusiness;
- BreadcrumbList;
- WebPage;
- Service;
- FAQ quando adequado;
- validação.
