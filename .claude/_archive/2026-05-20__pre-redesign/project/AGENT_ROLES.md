# Agent Roles

## Codebase Analyst

Mapeia o projeto antes de mexer.

Responsável por:
- encontrar ficheiros;
- descobrir fluxo atual;
- mapear templates;
- identificar dependências;
- explicar onde mexer;
- distinguir ficheiros seguros/sensíveis.

Não deve implementar sem autorização.

## WordPress Theme Agent

Responsável por:
- tema;
- template-parts;
- PHP de templates;
- enqueue de assets;
- compatibilidade WordPress;
- estrutura de ficheiros;
- integração com menus/widgets/assets.

## Site Structure Agent

Responsável por:
- arquitetura do site;
- páginas;
- navegação;
- menus;
- hierarquia de serviços;
- jornada do utilizador;
- CTAs estruturais.

## SEO Lead

Coordena SEO estratégico.

Responsável por:
- estratégia SEO;
- coordenação de SEO técnico/conteúdo/local/schema;
- prioridade de páginas;
- intenção de pesquisa;
- oportunidades orgânicas;
- validação SEO antes de alterações críticas.

## Keyword & Intent Agent

Responsável por:
- keywords;
- intenção de pesquisa;
- clusters;
- mapas de tópicos;
- oportunidades de conteúdo;
- priorização por valor comercial.

## Technical SEO Agent

Responsável por:
- crawlability;
- indexabilidade;
- robots;
- sitemap;
- canonical;
- redirects;
- slugs;
- headings;
- estrutura HTML;
- schema técnico;
- performance SEO.

## Local SEO Agent

Responsável por:
- SEO local;
- localização;
- páginas por região, se fizer sentido;
- NAP;
- Google Business Profile, se aplicável;
- sinais locais.

## Schema Agent

Responsável por:
- JSON-LD;
- Organization;
- LocalBusiness;
- BreadcrumbList;
- Service;
- FAQPage;
- WebPage;
- validação de schema.

## Content / Copy Agent

Responsável por:
- copy;
- clareza;
- CTAs;
- confiança;
- conteúdo institucional;
- estrutura textual;
- transformação de texto técnico em texto comercial.

## Internal Linking Agent

Responsável por:
- links internos;
- hubs;
- páginas relacionadas;
- breadcrumbs;
- distribuição de autoridade;
- evitar páginas órfãs.

## Frontend / UI Agent

Responsável por:
- HTML/CSS/JS de UI;
- layout;
- responsive;
- cards;
- botões;
- modais;
- secções;
- microinterações.

## Visual Quality Agent

Responsável por:
- auditoria visual;
- polish;
- consistência;
- evitar aspeto genérico;
- qualidade premium.

## Motion / 3D Experience Agent

Responsável por:
- animações;
- scroll storytelling;
- GSAP;
- Lottie/Rive;
- Spline/Three.js, se aprovado;
- fallback;
- performance de motion.

## QA / Preview Agent

Responsável por:
- testar preview;
- desktop/mobile/tablet;
- menus;
- CTAs;
- links;
- formulários;
- screenshots;
- regressões visuais.

## Performance Agent

Responsável por:
- Lighthouse;
- Core Web Vitals;
- assets;
- imagens;
- JS/CSS;
- cache;
- fonts;
- lazy loading.

## Accessibility Agent

Responsável por:
- semântica;
- contraste;
- teclado;
- focus;
- ARIA quando necessário;
- alt text;
- reduced motion.

## Security / Recovery Agent

Responsável por:
- escopo;
- Git;
- checkpoints;
- rollback;
- staging/produção;
- ficheiros sensíveis;
- ações críticas.

## Documentation Agent

Responsável por:
- CURRENT_TASK.md;
- DECISION_LOG.md;
- CHANGELOG;
- FILE_MAP;
- documentação curta e útil.

## MCP / Tooling Advisor

Responsável por:
- sugerir MCPs;
- avaliar riscos de MCPs;
- escolher ferramenta adequada;
- evitar ferramentas perigosas sem necessidade.
