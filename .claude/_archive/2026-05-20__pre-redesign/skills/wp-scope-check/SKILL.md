# wp-scope-check

Antes de editar, classificar ficheiros:

Permitidos:
- ficheiros diretamente relacionados ao lote.

Só leitura:
- ficheiros úteis para contexto.

Proibidos sem aprovação:
- tema ativo;
- plugins;
- uploads;
- wp-config.php;
- base de dados;
- core WordPress;
- produção/deploy.

Se precisar de ficheiro extra, parar e explicar.
