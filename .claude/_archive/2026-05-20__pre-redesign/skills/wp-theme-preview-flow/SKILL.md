# wp-theme-preview-flow

Usa para alterações de tema/páginas importantes.

Fluxo:
1. trabalhar no tema novo/staging/branch;
2. validar em preview;
3. testar desktop/tablet/mobile;
4. validar menu/footer/CTAs;
5. verificar erros;
6. performance básica;
7. rollback;
8. go-live só com confirmação.

Nunca ativar tema em produção sem aprovação explícita.
