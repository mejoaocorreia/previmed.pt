# technical-seo-crawl-audit

Usa para auditoria técnica.

Verificar:
- status codes;
- robots;
- sitemap;
- canonical;
- redirects;
- noindex;
- duplicate content;
- headings;
- broken links;
- orphan pages;
- schema;
- mobile;
- rendering;
- performance.

Output:
- problema;
- URLs;
- impacto;
- prioridade;
- correção recomendada;
- validação.
