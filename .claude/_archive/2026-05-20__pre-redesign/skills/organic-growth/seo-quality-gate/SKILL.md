# seo-quality-gate

Usa antes de aprovar alterações SEO importantes.

Bloquear se:
- noindex/canonical errado;
- H1 ausente/duplicado;
- title/meta ruins;
- conteúdo genérico;
- schema enganador;
- performance degradada;
- mobile quebrado;
- links críticos partidos;
- slug alterado sem redirect;
- plugin SEO conflituante.

Output:
- aprovado / bloqueado;
- motivos;
- correções necessárias.
