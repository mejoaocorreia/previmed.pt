# gsc-ga4-analysis

Usa esta skill quando houver Search Console/GA4.

Search Console:
- queries;
- pages;
- clicks;
- impressions;
- CTR;
- position;
- device;
- country;
- date comparison.

GA4:
- organic sessions;
- engagement;
- conversions;
- landing pages;
- events;
- device/country.

Regras:
- distinguir GSC e GA4;
- usar períodos comparáveis;
- explicar limitações;
- não somar dados agregados indevidamente.
