# competitor-gap-analysis

Usa esta skill para análise de concorrência.

Passos:
1. definir queries;
2. pesquisar SERP/localização;
3. identificar concorrentes orgânicos;
4. classificar tipo de resultado;
5. analisar estrutura, conteúdo, UX, schema e CTA;
6. identificar lacunas;
7. propor como superar.

Output:
- tabela de concorrentes;
- padrões da SERP;
- lacunas;
- oportunidade;
- recomendação.
