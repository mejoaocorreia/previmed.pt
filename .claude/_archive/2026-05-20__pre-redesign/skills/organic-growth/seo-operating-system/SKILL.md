# seo-operating-system

Usa esta skill quando o pedido SEO for amplo, estratégico ou envolver várias frentes.

Deves:
1. definir objetivo;
2. identificar dados disponíveis;
3. escolher agentes necessários;
4. separar estratégia, técnico, conteúdo e medição;
5. propor fases;
6. definir KPIs;
7. indicar dependências;
8. recomendar primeiro lote.

Output:
- diagnóstico;
- plano por fases;
- agentes necessários;
- ferramentas/MCPs;
- riscos;
- próximos passos.
