# keyword-cluster-map

Usa esta skill para criar clusters de keywords.

Campos:
- cluster;
- intenção;
- keyword principal;
- variações;
- página alvo;
- tipo de conteúdo;
- prioridade;
- notas;
- risco de canibalização.

Não priorizar só por volume.
Priorizar fit com negócio + intenção + oportunidade.
