# serp-intent-audit

Usa esta skill para perceber intenção real por query.

Verificar:
- tipos de páginas no top results;
- SERP features;
- local pack;
- PAA;
- snippets;
- vídeos/imagens;
- intenção dominante;
- formato esperado;
- profundidade de conteúdo.

Output:
- query;
- intenção dominante;
- página recomendada;
- formato;
- requisitos de conteúdo.
